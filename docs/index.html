// Features

Everything you need
to run your factory

Factorio Forge is built for players who want simplicity without sacrificing power.

One-Click Install

Download the installer, run it, and you're done. No configuration files, no command line — Factorio Forge just works out of the box.

Multiple Instances

Run different Factorio versions or mod setups side by side. Switch between a vanilla playthrough and a heavily modded one in two clicks.

Export & Import Packs

Bundle your entire modpack into a single shareable file. Drop a link in Discord and your friends are up and running with the exact same setup.

Community Sharing

Share modpacks with your community, clan, or the entire Factorio world. Import a pack with one click and start playing immediately.

Mod Management

View, enable, and disable mods per instance. Each instance has its own isolated mod folder so nothing bleeds between saves.

Open Source

Factorio Forge is completely free and open source. Inspect the code, contribute features, or fork it for your own launcher. MIT licensed.

// How it works

Up and running in
under a minute

01

Download Factorio Forge

Grab the latest release from GitHub. Available as a simple installer for Windows, macOS, and Linux.

02

Add your Factorio instance

Point Forge to your existing Factorio install, or create a fresh custom instance with the + button.

03

Manage mods & saves

Each instance has isolated mods and saves. Enable, disable, or add mods without touching other setups.

04

Export & share your pack

Hit "Export pack" to generate a shareable bundle. Send it to friends — they import it and play instantly.
